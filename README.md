# OnChainVault

## Project Description
OnChainVault is a simple and secure Ethereum-based vault smart contract that enables users to deposit and withdraw ETH. The contract records user balances and ensures that users can only withdraw their own funds.

## Project Vision
The vision for OnChainVault is to serve as the foundation for decentralized financial storage—simple today, but expandable into a full smart-contract-based savings and asset-management platform in the future.

## Key Features
- **Secure Deposits:** Users can safely deposit ETH into the contract.
- **User-Specific Balances:** Each user’s balance is tracked individually with full transparency.
- **Safe Withdrawals:** Only the rightful owner can withdraw their funds.
